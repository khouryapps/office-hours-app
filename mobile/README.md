### Setup 

1. move to the mobile directory
```shell
cd mobile
```

2. Install node modules
```shell
npm install
```

3. Make sure the simulator is app is installed

### Run

1. Run the simulator
```shell
react-native run-ios
```

2. Build the Java Script Bundel 
```shell
react-native start
```

3. Refresh the Simulator
```shell
cmd r
```
